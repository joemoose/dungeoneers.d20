TYROS – Players Guide (working title)
---

*TYROS*  
*Playtest Document v0.3, rules-lite edition*  
*February 21, 2023*  

TYROS is a rules-lite system for running old-school fantasy role-playing games.

- Tyros is inspired by Yochai Gal’s [Cairn](https://cairnrpg.com) and Ben Milton’s [Knave](https://questingbeast.itch.io/knave) and [Maze Rats](https://questingbeast.itch.io/maze-rats).
