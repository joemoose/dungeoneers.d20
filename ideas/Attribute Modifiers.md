# Attribute Modifiers

| Score |	Modifier |
| :-: | :-: |
| 1-3	| -3 |
| 4-5	| -2 |
| 6-8 | -1 |
| 9-12 | 0 |
| 13-15 | +1 |
| 16-17 | +2 |
| 18 | +3 |

## Attribute Saves

Roll 1d20 +MOD on risky or uncertain outcomes:

- **15+**: success
- **10-14**: success, with a complication
- **9-**: failure

## Crits

- **20** = critical success
- **1** = critical failure
